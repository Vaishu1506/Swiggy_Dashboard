# Swiggy_Dashboard
🍽️ Swiggy Sales Analysis Dashboard

A comprehensive Power BI Sales Analysis Dashboard built using a Swiggy-inspired dataset to analyze sales performance, customer ratings, food categories, and order trends. This dashboard provides actionable business insights through interactive visualizations and KPIs.

📌 Project Overview

The Swiggy Sales Analysis Dashboard helps monitor restaurant sales performance, customer satisfaction, and ordering trends. It enables users to filter data by Month, Food Category, and Restaurant Name to analyze business performance across different dimensions.
